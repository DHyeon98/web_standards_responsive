웹표준이란

브라우저 종류 또는 버전의 차이가 있어도 호환이
가능하도록 제시된 표준입니다.

웹표준의 장점

  1. Step1

    수정 및 운영관리 용이

    올바른 구조화와 CSS 표현 통일화로
    인한 수정 및 관리 용이

  2. Step2

    검색엔진 최적화

    구조화된 웹페이지는 검색엔진에
    효율적 노출될 가능 높음

  3. Step3

    호환성 가능

    웹페이지 구조화시 다양한
    브라우저에서 호환 가능

웹접근성이란

장애가 있는 사용자를 포함한 모든 사용자가
쉽고 효과적으로 상호작용 가능한 웹을 만드는
관행을 말합니다.

  1. Step1

    적절한 대체 텍스트

    시각적으로 인식하지 못하는
    사용자 또는 상황을 대비

  2. Step2

    자동 재생 금지

    자동으로 소리가
    재생되면 안 됨

  3. Step3

    키보드 사용 보장

    모든 기능은 키보드만으로
    사용할 수 있어야 함

  4. Step4

    깜빡임과 번쩍임 제한

    초당 3~50회 주기로 깜빡,
    번쩍거리는 콘텐츠 금지

  5. Step5

    기본 언어 표시

    주로 사용하는 언어를
    명시해야 함

  6. Step6

    사용자 요구에 따른 실행

    사용자가 의도하지 않는
    기능은 실행 금지
